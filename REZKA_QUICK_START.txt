🚀 3-STEP SETUP:

1. RELOAD THE EXTENSION (IMPORTANT!)
   
   chrome://extensions/
   ↓
   Find "Subtitle Overlay Connector"
   ↓
   Click refresh button (⟳)

2. START THE PYTHON APP
   
   python app/main.py

3. GO TO REZKA AND TEST
   
   https://rezka.fi/films/fantasy/107-hobbit-nezhdannoe-puteshestvie-2012-latest.html
   ↓
   Press Ctrl+F5 (reload page)
   ↓
   Play video with subtitles ON
   ↓
   Subtitles appear in floating window! 🎉

📋 VERIFICATION CHECKLIST:

Before testing on Rezka:
□ Extension reloaded in chrome://extensions/
□ Python app running (see "WebSocket server started" message)
□ System tray icon visible

On Rezka page:
□ Page reloaded (Ctrl+F5)
□ Click extension icon → Shows "✓ Current page is supported"
□ Click extension icon → Shows "✓ Connected to desktop app"

Playing video:
□ Video is playing
□ Subtitles enabled on Rezka player
□ Floating window visible
□ Subtitles appearing in floating window!

🔧 TROUBLESHOOTING:

❌ "Not connected" on Rezka:
   → Reload extension: chrome://extensions/ → refresh (⟳)
   → Reload Rezka page: Ctrl+F5
   → Test: Click extension icon → "Test Connection"

❌ No subtitles appearing:
   → Make sure subtitles are ON in Rezka player
   → Press F12 → Console tab → Look for "Rezka subtitle detector activated"
   → See REZKA_SETUP.md for advanced troubleshooting
